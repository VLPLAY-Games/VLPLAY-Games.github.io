Описание

Легковесный музыкальный проигрыватель для Windows для локальных аудиофайлов (размером менее 1 МБ). Компактное и эффективное приложение для прослушивания музыки с вашего компьютера.

Основные возможности

📦

Компактный размер

Всего ~1 МБ общий размер

🎵

Поддержка форматов

MP3, WAV, FLAC, AAC, OGG, WMA, M4A

📊

Информация о треках

Длительность, битрейт, расширение, размер, дата создания

⏯️

Управление воспроизведением

Play/Pause, Следующий, Предыдущий трек

🌐

Мультиязычность

Поддержка русского и английского языков

🔄

Режимы воспроизведения

По порядку, Случайно, Повтор

⏱️

Расчет длительности

Общая длительность всех треков

📁

Сканирование папок

Отображение статуса сканирования папок

Установка

1

Скачайте последний релиз

Перейдите в раздел Releases на GitHub

2

Разархивируйте ZIP файл

Извлеките содержимое архива

3

Запустите приложение

Запустите файл MusicLocalUI.exe

Включенные файлы:

  • MusicLocalUI.exe - Основное приложение
  • TagLibSharp.dll - Библиотека метаданных

Как использовать

1

Нажмите "Open" чтобы выбрать папку с музыкой

2

Нажмите "Scan" чтобы загрузить треки

3

Используйте элементы управления воспроизведением:

  • ▶️ Play/Pause текущего трека
  • 'Next' для следующего трека
  • 'Previous' для предыдущего трека
4

Просматривайте информацию о треках в разделе "Track info"

Устранение неполадок

Если приложение не запускается:

  1. Установите .NET Framework 4.7.2
  2. Убедитесь, что оба файла находятся в одной папке
  3. Запустите от имени Администратора при необходимости
