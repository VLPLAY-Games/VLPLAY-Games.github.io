Описание

Yuki Protocol — унифицированный протокол обмена данными версии 1.1, разработанный для обеспечения единого формата взаимодействия между всеми компонентами экосистемы.

Протокол реализован на четырёх языках: Python (Core и Web UI), JavaScript (Web UI), C# (Windows Client) и Kotlin (Android Client). Это позволяет устройствам и клиентам обмениваться сообщениями в едином формате независимо от платформы.

Типы сообщений

🤝

Управление подключением

hello, welcome, ping, pong

📊

Статусы и метрики

status, extended_status, metrics

📨

Команды

command, command_result, confirm_command

🔐

Авторизация

🔑

Токены

token_update, token_info, rotate_token

🔗

Связь устройств

device_to_device, device_response, device_broadcast
