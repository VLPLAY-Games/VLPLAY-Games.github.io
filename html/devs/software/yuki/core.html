Описание

Yuki Core — центральный сервер экосистемы Yuki. Обеспечивает управление устройствами, аутентификацию, маршрутизацию команд, сбор метрик и ведение логов. Реализован на Python с использованием Flask для веб-интерфейса и WebSockets для реального времени.

Core поддерживает подключение неограниченного количества устройств, управление через Web UI, авторизацию устройств, ротацию токенов, чёрные списки, аудит действий и сбор системных метрик.

Ключевые возможности

🔌

Управление устройствами

Подключение, авторизация, чёрные списки, статусы онлайн/офлайн

📨

Маршрутизация команд

Отправка команд, широковещательные рассылки, связь устройство–устройство

📊

Метрики и логи

Системные метрики, аудит событий, история команд, uptime статистика

🔐

Безопасность

Аутентификация по токену, ротация токенов, чёрные списки, rate limiting

📱

API для клиентов

WebSocket API для устройств и клиентов, REST API для Web UI

🗄️

Хранение данных

SQLite база данных для устройств, команд, метрик и аудита
