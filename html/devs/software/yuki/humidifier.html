Описание

Yuki Humidifier — референсное IoT-устройство на базе микроконтроллера ESP32-C3 Super Mini. Это умный увлажнитель воздуха с полной поддержкой Yuki Protocol v1.1.

Устройство подключается к Core через WebSocket, поддерживает авто-режим, управление через веб-интерфейс, сбор метрик, обработку команд и взаимодействие с другими устройствами экосистемы.

Возможности

💨

Управление увлажнением

Включение/выключение, установка целевой влажности

🤖

Авто-режим

Автоматическое поддержание заданного уровня влажности

🌐

Веб-интерфейс

Встроенный веб-сервер для локального управления

📡

Связь с Core

WebSocket подключение, авторизация по токену

📊

Метрики

Текущая влажность, состояние, uptime, CPU частота

🔗

Взаимодействие

Обмен командами с другими устройствами экосистемы

Аппаратная платформа

🔌
ESP32-C3 Super Mini RISC-V, 160 MHz
⚡
Пины GPIO1 (вентилятор), GPIO21 (увлажнитель)
💾
Память 4MB Flash, SPIFFS для хранения конфигурации
📡
Связь Wi-Fi 802.11 b/g/n
